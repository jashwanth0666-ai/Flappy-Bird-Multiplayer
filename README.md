# 🎮 Flappy Bird Multiplayer

A real-time multiplayer Flappy Bird game built using **HTML, CSS, JavaScript, Node.js, and Socket.IO**.
Players can create or join rooms and compete live with synchronized gameplay.

---

## 🚀 Live Demo

👉 https://flappy-bird-multiplayer-rctr.onrender.com

---

## ✨ Features

* 🎮 Classic Flappy Bird gameplay
* 🌐 Real-time multiplayer using WebSockets
* 🏠 Room-based system (create/join rooms)
* 🐦 Multiple players with live movement sync
* 🧱 Server-controlled pipes (shared obstacles)
* 📊 Live scoreboard
* 👤 Player identity system
* 🎨 Modern UI with animations
* 📱 Mobile-friendly controls

---

## 🧠 Tech Stack

### Frontend

* HTML5 Canvas

### Backend

* Node.js

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/jashwanth0666-ai/Flappy-Bird-Multiplayer.git
cd flappy-multiplayer
```

### 2️⃣ Install Dependencies

```
npm install
```

### 3️⃣ Run Locally

```
npm start
```

Open:

```
http://localhost:3000
```

## 🎯 How to Play

1. Enter your name
2. Create or join a room
3. Press **Space / Tap** to fly
4. Avoid pipes and survive
5. Compete with other players in real-time
